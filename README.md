## 2. Short Description / Purpose  

The **Weather_Forecast Dashboard** is a live Power BI report that consumes data from a weather API to present real-time weather conditions and short-term forecasts.  
It enables users to monitor temperature, humidity, wind, pressure, and weather conditions in a clear, interactive, and visually intuitive format.

## 4. Data Source  

**Primary Source:** Weather API (Live REST API)

### Data Includes:
- Real-time temperature, humidity, wind speed, and pressure  
- Weather condition descriptions (sunny, cloudy, rain, storm)  
- Hourly / daily forecast data  
- City-level and geographic metadata  

### Data Structure:
- Current Weather Table  
- Forecast Weather Table  
- Location Metadata Table

## 5. Features / Highlights  

### Business Problem  

Access to accurate and real-time weather information is critical for individuals, businesses, and operational planning.  
However, raw API responses are technical and difficult to interpret without visualization.

Challenges include:
- Lack of real-time analytical dashboards  
- Difficulty extracting insights from JSON-based API data  
- No centralized platform for weather trend analysis  

---

### Goal of the Dashboard  

The **Weather_Forecast Dashboard** aims to:

- Provide real-time weather visibility in a single interface  
- Convert raw API data into meaningful insights  
- Support travel planning, operational decisions, and monitoring  
- Demonstrate real-time analytics capabilities using Power BI  

---

### Walkthrough of Key Visuals  

#### Key KPIs  
- Current Temperature  
- Feels-Like Temperature  
- Humidity (%)  
- Wind Speed  
- Atmospheric Pressure  
- Current Weather Condition  

#### Live Weather Status Panel  
- Current weather type (Sunny, Cloudy, Rainy, Stormy)  
- Visibility level  
- Day/Night indicator  

#### Temperature Trend (Line Chart)  
- Shows forecasted temperature changes  
- Helps identify warming or cooling patterns  

#### Humidity & Wind Analysis  
- Comparative analysis of humidity and wind speed  
- Supports comfort and safety evaluation  

#### Forecast Summary  
- Hourly or daily forecast view  
- Expected temperature ranges and weather conditions  

#### Location Filter  
- City-based slicer  
- All visuals update dynamically based on selection
